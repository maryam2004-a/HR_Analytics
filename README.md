# HR Attrition Analysis Dashboard

Power BI project analyzing HR attrition data to identify patterns and insights.

## Features
- Dashboard with KPIs, Attrition vs Overtime, Salary vs Attrition, Job Role analysis, etc.
- Interactive slicers for Age Group, Education Level, and Department
- Cleaned and preprocessed HR dataset

## Tools Used
- Power BI
- DAX Measures for Attrition Analysis
- Python for data preprocessing (if used)

## Data
Dataset used: WA_Fn-UseC_-HR-Employee-Attrition.csv (public dataset)
